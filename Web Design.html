WHAT IS WEB DESIGN

Webdesign refers to the design of website that are displayed o the internet. it usually refers to the user experience aspects of website development rather than software development. Webdesign used to be focused on designing website for desktop browsers; however, since te mid 2010's, desig for mobile and tabket browsers has become ever-increasingly important.A Webdesign works appearance layout,and, in some cases, content of a website. Appearance, for instance, relates to the colors , font, and images used. Layout refers to how information is structed and categorized. A good web design is easy to use, pleasing and suits the user group and brand of the website.

  • HTML

    • HTML stands for Hyper Text Markup Language, it uses markup to describe the structure of webpages
    The first version of html was written by Tim Berners Lee in 1993.In the 2000's the most widely used version was HTML 4.01, which became an official standard in 1999.Another version XHTML was a rewrite of html as an XML language.


    Uses Of HTML

  • HTML is used for structuring webpages
  • For navigating the internet
  • For embedding images and videos
  • For improving client-slide data storage and offline capabilities
  • For game development

    • Why HTML

    HTML code ensures the proper formatting of text and images for your internet browser. Without HTML a browser would not know how to display text as elements or load images or other elements

    Uses of CSS

    CSS makes the font-end of a website shine and it creates a great user exerience. Without CSS website would be less pleasing to the eye and likely much hadder to navigate.
